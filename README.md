## Featured Project

### [PrimeSec Infrastructure](https://github.com/r0drigosh/primesec-infrastructure)

Virtualized infrastructure lab built with Proxmox VE, OPNsense, Windows Server, Active Directory, Ubuntu Server, and Apache.

| Component | Role |
| --- | --- |
| `FW-01` | Firewall, NAT, DHCP, and Tailscale remote access |
| `DC-01` | Active Directory, DNS, and Group Policy |
| `WS-01` | Windows 11 Pro domain workstation |
| `WEB-01` | Ubuntu Server with Apache HTTP Server |

Includes architecture documentation, validation screenshots, and exported Group Policy reports.
